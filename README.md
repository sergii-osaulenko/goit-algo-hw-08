# goit-algo-hw-08
Homework #8

Код виконується і повертає мінімальну суму витрат за допомогою модуля heapq.
Алгоритм має складність O(n log n), де n - кількість кабелів.

Жадібний підхід - завжди з'єднуємо два найкоротші кабелі, бо їхня сума
додається до кожної наступної операції найменше разів.

Мінімальна купа (priority queue) забезпечує швидкий доступ до найменших
елементів.
