# goit-algo-hw-08
Homework #8

Код виконується і повертає мінімальну суму витрат за допомогою *heapq. Алгоритм має складність O(n log n), де n - кількість кабелів.
​
# Пояснення алгоритму
Жадібний підхід - завжди з'єднуємо два найкоротші кабелі, бо їхня сума додається до кожної наступної операції найменше разів. Мін-купа heapq забезпечує швидкий доступ до найменших елементів.
